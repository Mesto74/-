特点类--概括特点

# 一、题型识别

（1）常见问法：特点、特征......

（2）典型例题

【2020 广东乡镇】请你根据给定材料，谈一谈乡镇公务员工作的特点。（15 分）要求：紧扣材料，符合实际，表达清晰；篇幅不超过 200 字。

【2017 上海 B】结合给定资料，谈谈未来中国制造应具备的主要特征。(20 分) 要求：概括准确，条理清楚，语言精练，字数不超过 300 字。

# 二、基础理论

# （一）小众要素处理方式

  1. 转化为常规要素例：请概括自己为了上岸做出的努力。

  2. 利用常识进行预判——是什么；标志词；答案呈现形式

例：“变化” --> 关注时间或其他的事件带来的改变

# （二）什么是特点

例：小韩的特点是什么？

材料：小韩不仅肤白貌美大长腿，而且毕业于中国人民大学，是妥妥的高材生；她平时喜欢品鉴甜品，定期健身，画起画来也是一把好手。

答：外形姣好，成绩优异，喜欢甜品、健身，绘画好。

特点是对特定事物（主题）在某个角度的抽象性（总括性）描述，表达形式灵活多样。

# （三）特点类题目的答题技巧

1.主题词定位法

第一步：定位题干主题词

第二步：在资料中主题词附近找描述性语言

例：请概括沈浩的特点。

材料：选派优秀机关干部到村任第一书记，是加强农村基层党组织促脱贫攻坚的重要举措，这一举措已经实施多年。在我国著名的小岗村，就有一位让人难以忘怀的第一书记。2004 年，40 岁的沈浩来到这里之后，带领村民修路、打井，支持青年创业致富……

答：沈浩躬身践行，带领村民修路打井，支持创业。

2.标志性词汇

（1）形容性质的词汇：高、低、快、慢；XX性、XX 感、XX 化

（2）对比词：更加、更；比一般、和过去相比、和......相比；区别、亮点、不同

# （四）特点的表达形式

例：小韩小学时语文数学就常常考100分，后来她中考考进了重点中学，经过三年苦读取得了647分的高考成绩，后又以390 分的研究生初试成绩进入中国人民大学读研。

问：小韩的特点是什么？

答：成绩优异。小学主科满分，中考进入重点中学，高考和考研分数高。

总结性特点 + 具体描述性信息。

# （五）特点的特殊表达形式

例：根据“给定资料1”，概括浙江四条诗路各自的特点。（10 分）

【参考答案】

1.浙东唐诗之路：名山众多，吸引文化名人，兴起士文化；名寺古刹兴盛，融合佛教道教。

2.大运河诗路：文化遗产占比高，覆盖历史文化名城，串联水乡古镇，体现江南千年古韵；反映丝路盛况。

3.钱塘江诗路：展示钱塘江大潮奇观；包孕“弄潮儿”精神，形成著名文化学派。

4.瓯江山水诗路：孕育山水诗派，展示传统文化生活及技艺；体现人与自然和谐相处的文化内涵。

# 三、例题演练

根据给定资料 1，概括新凤村“土专家”的特点。（30 分）要求：全面、准确、简明，有条理，字数不超过200 字。

【给定资料】

走进新凤村，这里林木环绕、空气清新，道路整洁，民居井然有序，但在四年前，新凤村还是个破败萧条的贫困村。这个村为何能在这么短的时间里发生如此大的变化呢？笔者满怀好奇地向当地的同志问道：“新凤村的喜人局面从何而来？”对方笑答，因为这里有一批“土专家”，他们都是土生土长的当地人，对社情民意清楚得很，这些“土专家”在脱贫攻坚中结合新凤村的实际，干了不少专业的事。

要点：土生土长。熟悉社情民意，结合当地实际开展专业工作。
