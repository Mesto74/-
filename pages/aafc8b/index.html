影响类--概括影响

# 一、识别影响

# 什么是影响？

影响指某个事情发生后，带来的一系列变化，是在事后的结果。

# 影响的分类

正面影响的常见问法：意义、效果、成果、积极作用等。

【例 1】（2019 广东县级）请根据材料6，分析说明 A 市（灯饰）知识产权快速维权中心在促进产业发展方面取得的成效。

【例 2】（2020 江西乡镇）根据给定资料2，请谈谈建立基层立法联系点制度在该市发挥了哪些积极作用？

负面影响的常见问法：危害、后果等。

【例 1】（2017 四川上半年）根据给定资料2，概括当前网络伦理失范的危害。

【例 2】（2016 黑龙江B 卷）请你根据给定资料 3，概括说明非法采砂的危害。

# 总结

影响指某个事情发生后，带来的一系列变化，是在事后的结果。

（1）正面影响的常见问法：意义、效果、成果、积极作用等。

（2）负面影响的常见问法：危害、后果等。

# 二、答题技巧

# 1.寻找与主题相关的标志性词汇

①积极影响的标志性词汇：促进了、改善了、增加了、提高了、减轻了、实现了、弘扬了、节约了、保护了、方便了、倒逼......

【例 1】政府加大处罚力度，倒逼企业减少污染，加强环境环保。

②消极影响的标志性词汇：降低了、减少了、破坏了、阻碍了、引发了、加重了、激化了......

【例 1】暴饮暴食加重了他的病情，降低了他完全康复的概率。

③中性标志性词汇：引发……，导致……，造成……，致使……等，要结合材料具体信息点进行判断。

【例】这些措施使得农村面貌发生了变化。

# 2.分析推导

①通过问题进行推导：资料中想通过某项举措去解决问题，或者问题已经被措施解决了，可以将问题推导为相关做法的意义。

某地工业发展落后，产业结构不科学，针对这一问题，当地政府出台相关的扶持政策。

政府出台的相关扶持政策 帮助改善当地工业落后的发展面貌，科学优化产业结构

某地工业发展落后，当地领导班子经过研究，重点发展生态产业和休闲旅游产业。

这种的就推不出来

②通过目的进行推导：材料中出现表示目的或目标的信息时，可推导为意义。常见的表目的和目标的提示词有：为了……，以……为目标，意在……，本项目以……为主旨......

为了提升申论成绩，我坚持听课和练习。

坚持听课和练习的意义是 为了提升申论成绩

# 三、例题详解

【例 1】根据给定资料，请谈谈信用体系建设对当前社会有哪些积极影响？

【材料节选】2018 年，小张到A市工作，在租房时，她惊喜地被中介告知，由于她的信用记录良好，所以不用缴纳两倍于月租的租房押金。同在A市的小李到当地妇女儿童医疗中心就诊时发现，因为自己的信用度足够好，在医疗中心就可以“先诊疗后付费”，而无需一次次在诊室、检查室和交费处之间奔波，大大节省了自己的时间和精力。

【要点总结】方便民众的生活。节省时间、精力、减轻生活负担。

【例 2】根据给定资料，概括说明非法采砂的危害。

【材料节选】非法采砂船为了多采砂，不分地点，哪里有好砂就在哪里下泵抽砂，致使国家投入巨资兴建的丁坝、护岸等航道整治建筑物地基被掏空、垮塌和损毁。如长江干线戴家洲水道，国家投资2亿多元建设的航道护坡护岸工程，被非法采砂挖得千疮百孔；2013 年初，长江下游九江水道鳊鱼滩航道整治工程被 3 艘挖砂船近距离非法采砂破坏，国家耗资数百万元才得以修复。

【要点总结】破坏基础设施，造成经济损失。航道整治建筑物地基被掏空、垮塌和损毁，护坡护岸工程被挖，浪费国家财力。
