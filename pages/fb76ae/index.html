顺序逻辑

# 一、资料逻辑有多重要？

【2022 国考行政执法卷】“给定资料 1”反映了N市积极落实惠企政策的有关情况，请简述其主要做法及成效。(15分)

要求：全面、准确、有条理。不超过250字。

【例二】根据“给定资料2”，谈谈你对“现在撤掉的是‘眼中的柜台’，但我们更要在撤掉‘心中的柜台’上下功夫”这句话的理解。（15 分）

要求：分析全面，条理清晰。不超过300字。

【例三】J 市税务局积极优化税收服务，为群众办实事，采取了多项举措，效果良好，请根据“给定资料3”，就 J 市税务局如何进一步强化举措、巩固成果，形成长效机制，撰写一份工作建议。（20分）

要求：（1）紧扣资料，内容全面；（2）建议具有针对性、可行性；（3）准确简明，条理清晰；（4）不超过 500 字。

【例四】有关部门拟召开一场以“严执法，优服务，促发展”为主题的经验交流会。请根据“给定资料4”，撰写一份关于 L 海关的工作经验交流材料提纲。(20 分)

要求：（1）紧扣资料，内容全面；（2）准确简明，条理清晰；（3）不超过500 字。

从经验角度得出，小题两个重要的得分点：

  1. 要点分——内容全面，在尽量保证精准的前提下，宁滥勿缺。

  2. 逻辑分——① 答案条理清晰，重点突出；② 分类准确，每条答案的话题独立。

# 二、逻辑定义

# （一）材料逻辑

宏观逻辑：围绕材料话题的变化进行情况分类，体现在答案中是分类 1.2.3.

微观逻辑：要点的层级，体现在答案中为准确提取前置词

# （二）分类的要求

形式上：①分条列点；②每条前加总括词（句）

内容上：要点间内容不交叉，话题相互独立，呈并列关系

示例：

  1. 资金直达直兑。开设绿色通道，提前做好准备工作，及时高效精准分配资金；推出免申即享服务，增强企业获得感。

  2. 优化惠企服务。上门服务，现场解答、主动告知提醒政策红利；提供帮办代办服务，促使企业办事便捷高效舒心。

  3. 搭建智慧平台。推行掌上提、在线批、直达付、全程督服务模式，政策直达准快实；涉企部门入驻，提交服务清单，开展互动联建，迅速解决难题。

  4. 加强多元监督。通报违规案例，开展智慧监督，预防问题发生，政策落实公正透明高效，让企业安心。

  5. 实现异地易办。推出电子系统，政务服务跨省、一网通办，省时省力。

# 三、宏观逻辑-分类依据

顺序逻辑（块状逻辑）：每个段落/资料只有一个要点，且内容上各自独立，这时可按照自然段的顺序分条罗列要点，可能为一个段落/资料一个内容，也可能是几个段落/资料围绕一个内容展开。

乱序逻辑： 例如：第2段和第5段都在说产业问题，就要把他俩放在一个点里概括

# 四、微观逻辑-前置词作答方法

前置词—核心词——如何得出：

# 1. 总分关系

常规总分词，一般体现为资料中有总结句

① 内容——明确的总分关系：资料中关键词有确凿的从属关系

产业多元化、农业农产品加工、休闲旅游业，即产业多元化为前置词总括后面俩

② 位置——通常为首句、尾句。

首句、设问、尾句、概括...... 很多方式暗示出了前置词

③ 词汇特征——规范，但不具体，相对宏观

万众一心词：一般体现为多因一果、多果一因

【例】为了改善民生，N市政府采取了一系列举措，完善社会保障体系，提振经济增加就业，改进公共服务。 --> 改善民生

【例】N 市政府采取了一系列举措，完善社会保障体系，提振经济增加就业，改进公共服务，这些做法都极大地改善了民生 --> 改善民生

# 2. 高频词

文段中反复出现的词汇：同义词、同个词

【例】根据“给定资料”，概括F 市陶瓷产业发展受阻的主要原因。

【给定资料】

上世纪 80 年代，借着改革开放之风，F市“村村点火、户户冒烟”，陶瓷产业蓬勃发展起来。2005 年，F 市陶瓷迎来一场生死考验。国际油价节节攀升，对于燃油费用占总成本三成以上的陶瓷产业来讲，压力陡增；工人工资上涨，为节约人力成本生产线只能“开三停四”，更是雪上加霜。紧接着2010 年，土地成本上升波及陶瓷行业。这让利润可观的陶瓷行业整体行情陡转直下，这期间许多企业都因此停业。

【答】 生产成本上涨

# 3. 强调词

“是核心”、“是本质”、“是重点”、“最重要的是”......

【例】专家 D：实现农业农村现代化，也不能平均发力，而应抓重点、补短板、强弱项。经过多年努力，农村公共服务体系实现了从“无”到“有”的历史性转变。未来应将重点放在缩小公共服务领域的城乡差距，提高农村公共服务的保障水平上，要把从“有”到“好”作为主攻方向，继续推动城乡义务教育一体化发展，完善城乡居民基本养老保险制度，完善统一的城乡居民基本医疗保险制度和大病保险制度，统筹城乡社会救助体系。随着大量农村人口转移进城，部分地方出现了“空心屋”“空心村”，这是现代化进程中的必然现象，也是正常现象。我们要做的，是为乡村聚人气、添活力。

【答】 缩小公共服务领域的城乡差距，提高农村公共服务的保障水平

# 4. 逻辑承接词

（段首尾处）表并列的关联词句：除了......还.....；既.....又.....；不但.....而且.....；不仅......还.....

表达共性：除了以上做法，为了解决产业结构不合理的问题，A 市还积极与众多重点行业的龙头企业开展合作。

【答】 解决产业结构不合理的问题

承接逻辑：除了解决产业结构不合理的问题，B省还积极推动传统行业转型升级。

（提示性较弱）：也、又、还

# 5. 指代词

“这一”、“这个”、“其”、“此”

【例】根据给定资料，概括指出线下文艺的优势。

【给定资料】

在某书画院的李院长看来，“现场演出能够确保观众浸润在浓郁的文化氛围之中，观众或驻足凝望，或沉思遐想，或洗耳恭听，与他人作品背后的故事、情感、智慧等高度共情，进而产生一种参与感。” “虽然在网上也能看文艺表演，但是远不能收获那些美好体验。”

【答】线下体验更好

# 6. 根据自身储备归纳概括

【例】根据材料，对当前农村地区“乡村病”的具体表现进行概括。

村小组长介绍，800 多常住人口，差不多有400人在外务工，一幢幢200 多平方米的老屋仅住着祖孙三人或四人。还抱怨说：“我们组织办班培训，让中青年掌握一技之长，他们外出谋生没问题。可是，让他们在家门口就业就不是村里能做到的，留守儿童、空巢老人的问题也不好解决。”

【答案】空心化。1乡村就业难，2劳动力外出打工，3留守儿童、空巢老人多

# 总结

前置词—核心词——如何得出

① 总分关系：常规总分词、万众一心词、自身概括性较强的词

② 高频词：文段中反复出现的词汇或表意

③ 强调词：是核心、是本质、是重点

④ 承接词：承上启下

⑤ 指代词：这、那、此

⑥ 根据自身储备归纳概括：硬核储备
